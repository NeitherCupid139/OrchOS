# Third-Party Licenses

This directory collects license information for direct dependencies used by this repository.

- Project license: `../LICENSE`
- Dependency inventory: `./index.json`
- One text file per direct dependency is included here.

Notes:
- When an installed package includes a `LICENSE`, `LICENSE.md`, `LICENSE.txt`, `NOTICE`, or `COPYING` file, that file is copied into the corresponding entry here.
- When no standalone license file is present in the installed package directory, the entry records the package metadata license field and source path.

Generated from the currently installed direct dependencies in the root package and `apps/web/package.json`.
